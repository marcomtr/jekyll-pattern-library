Grid System

Bootstrap

Our grid system is based on Bootstrap 4, which includes several components and options for laying out your pages. You can create a basic grid system that starts out stacked on mobile devices and tablet devices (the extra small to small range) before becoming horizontal on desktop (medium) devices.
